# Suffolk Law Interactive Learning Tools

AI-powered interactive learning tools for Suffolk University Law School students.

## Tools

### 1. Bar Exam Question Bank
AI-generated MBE-style practice questions with detailed rationale explanations for every answer choice. Covers 7 subject areas: Contracts, Torts, Constitutional Law, Criminal Law, Property, Evidence, and Civil Procedure.

## Deployment
This project is deployed via Vercel and embedded in Suffolk Law's Canvas LMS.

## Tech Stack
- React 18
- Vite
- Anthropic Claude API
